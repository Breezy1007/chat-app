# دردشة - تطبيق شات (PWA) 💬

تطبيق شات بلا حدود، بلا Play Store، بلا حساب مطور، **بلا ما تخلص حتى درهم**.

---

## 1. كيفاش كيخدم

هاد التطبيق "PWA" (Progressive Web App) - يعني موقع كيتصرف بحال تطبيق حقيقي:
- الناس كيدخلو للرابط ديالو من المتصفح
- كيضغطو "إضافة إلى الشاشة الرئيسية" (Add to Home Screen)
- كيبان ليهم آيقونة بحال أي تطبيق آخر، بلوغو، وكيخدم بلا شريط المتصفح

---

## 2. الخطوة الوحيدة اللي خاصك: صيفط مشروع Firebase (مجاني 100%)

1. سير ل https://console.firebase.google.com
2. "Add project" → عطيه سمية → كمل
3. من الصفحة الرئيسية: دوز على "</> Web app" (إيقونة `</>`)
4. غادي يعطيوك كود فيه `firebaseConfig` (apiKey, projectId...)
5. افتح ملف `app.js` وبدل هاد السطور بالقيم ديالك:
```js
const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "...",
};
```
6. من القائمة الجانبية فـ Firebase: فعّل **Authentication** → Sign-in method → فعّل "Email/Password"
7. فعّل **Firestore Database** → Create database → Start in **test mode**

⏱️ هادشي كيدير تقريبا 10 دقايق، وماخاصكش بطاقة بنكية ولا فلوس.

---

## 3. فين تنشر التطبيق (استضافة مجانية 100%)

عندك بزاف ديال الخيارات المجانية، اختار اللي بغيتي:

### الخيار الأسهل: Firebase Hosting (مجاني، من نفس المشروع)
```bash
npm install -g firebase-tools
firebase login
firebase init hosting
# اختار المجلد الحالي، اختار "index.html" كـ public directory
firebase deploy
```
غادي يعطيك رابط بحال: `https://your-project.web.app`

### أو: Netlify (سهل بزاف، drag & drop)
1. سير لـ https://app.netlify.com/drop
2. سحب المجلد ديال المشروع (chatapp-pwa) وحطو فالصفحة
3. خلاص، غادي يعطيك رابط مباشرة

### أو: GitHub Pages (مجاني، إلا عندك حساب GitHub)
1. صيفط repository جديد وزيد فيه هاد الملفات
2. من Settings → Pages → فعّل GitHub Pages

**الثلاثة مجانيين 100%، ماكاينش أي فرق فالفلوس - اختار غير الأسهل بالنسبة ليك.**

---

## 4. كيفاش الناس يحملو التطبيق

منين يكون الرابط جاهز، شارك ليه مع صحابك:
- **من أندرويد (Chrome)**: كيبان زر "إضافة إلى الشاشة الرئيسية" أوتوماتيكيا
- **من آيفون (Safari)**: يضغطو على زر المشاركة (⬆️) → "إضافة إلى الشاشة الرئيسية"

بعد هادشي، التطبيق كيبان بحال أي تطبيق آخر فهاتفهم.

---

## 5. الاشتراك الشهري (Premium)

الزر ديال "اشترك" دابا كيبدل غير حالة فقاعدة البيانات (للتجربة). باش تاخد فلوس بجد بلا Play Store/App Store:

- **Stripe** (https://stripe.com) - كتقدر تدير صفحة دفع بسيطة (Stripe Checkout)، وكتاخد نسبة صغيرة غير من الفلوس اللي كتدخل - **بلا ما تخلص حتى درهم قبل**
- خاصك سيرفر صغير (Cloud Function مجانية من Firebase) باش يستقبل تأكيد الدفع من Stripe ويبدل `isPremium` بطريقة آمنة

إلا وصلتي لهاد المرحلة وبغيتي تربط Stripe، قوليا ونديروها سوا.

---

## 6. حوايج مهمة قبل ما تشارك التطبيق مع بزاف ديال الناس

- Firestore فـ "test mode" كيبقى مفتوح لمدة 30 يوم بعدها كيسد أوتوماتيكيا - خاصك تدير Security Rules قبل
- زيد **سياسة الخصوصية** بسيطة (صفحة وحدة كتشرح واش كتجمع من المعلومات)
- جرب التطبيق مزيان من هاتفك قبل ما تشاركو

---

صيفط ليا شي سؤال إلا وقفت فشي حاجة، أو بغيتي نزيد نربط Stripe 🚀
